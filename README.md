## rmf\_site\_map\_server

This package provides a ROS 2 node that publishes a site map generated by the [RMF Site Editor](https://github.com/open-rmf/rmf_site).

### Building

Make sure you have a colcon workspace with both [rclrs](https://github.com/ros2-rust/ros2_rust) and [rmf_building_map_msgs](https://github.com/open-rmf/rmf_building_map_msgs/), as well as the [colcon_cargo](https://github.com/colcon/colcon-cargo) and [colcon_ros_cargo](https://github.com/colcon/colcon-ros-cargo) extensions.
Afterwards rebuild your workspace and run the node:

```
colcon build --packages-up-to rmf_site_map_server
source install/setup.bash
ros2 run rmf_site_map_server site_map_server path_to_site_file.site.json
```
